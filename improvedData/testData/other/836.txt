Date Tue Dec GMT Server NCSA Content type text html Grail Local Interest Grail Disk Usage Policy Due the tendency hard disks fill without countermeasures taken the Grail group has adopted Disk Usage Policy This policy not intended inconvenience users who legitimately need the space Instead intended ensure that space available for those very users when they need Generally all the specific policies outlined below fall under the common sense rule using only what you need and being considerate others needs That being said here are the guidelines Students Students fall under the common sense rule you need more than megabytes consider whether you should using project space instead personal space And please remember remove large files when they longer needed Projects Projects also fall under the common sense rule Some will get very large Try limit extreme usage short periods possible you need large amounts space for significant amounts time please inform the sysadmin Graduated Students Students who have graduated from the program should determine which files their personal directory should made public and permanent and should install these files public permanent location See the sysadmin for how this Any remaining files should total less than ten megabytes and these will kept for six months following the student graduation Expired Accounts Files belonging expired accounts will kept for two weeks and then removed Last updated July
