theory Hybrid Knowledge Bases due Nerode and Subrahmanian this framework external information sources are abstracted domains which execute certain functions with pre specified input and output type These domains are accessed mediators using logic based declarative language This language based Annotated Logics due Kifer and Subrahmanian and provides powerfull and extensible programming environment The system also provides uniform environment for the easy addition new external sources existing mediators The system currently runs Sun Sparc stations under Unix well the IBM platform
